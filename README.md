# Lustful Spirit Hunt 中文汉化补丁

> **补丁版本：** V0.4
>
> **项目地址：** [GitHub](https://github.com/nekocatso/Lustful-Spirit-Hunt-Chinese)

## 安装方法

1. 找到游戏安装目录下的 `StreamingAssets` 文件夹，路径通常为：

   ```
   Lustful Spirit Hunt\Lustful Spirit Hunt_Data\StreamingAssets
   ```

   > 路径会根据你的安装位置变化，只需找到游戏目录下 `Lustful Spirit Hunt_Data\StreamingAssets` 即可。

2. 将本项目中所有 `.csv` 文件复制到该目录，覆盖原文件。

3. 启动游戏即可。

## 反馈与协作

- 游戏版本更新导致翻译失效，或者术语需要修正，欢迎在 [GitHub Issues](https://github.com/nekocatso/Lustful-Spirit-Hunt-Chinese/issues) 提交反馈
- 翻译改进欢迎提交 [Pull Request](https://github.com/nekocatso/Lustful-Spirit-Hunt-Chinese/pulls)

## 项目结构

- `glossary.py` — 术语库
- `*.csv` — 原始英文文件
- `game-translation-design.md` — 翻译设计参考文档
